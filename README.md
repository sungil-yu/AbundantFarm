# Project-AbundantFarm

은풍한 farm 프로젝트는 생산자의 생산 농지에서 농산물을 직접 소비자에게 전달해주는 서비스 입니다.

개발환경은 아래와 같습니다. <br/>
 - Language: Java, Html, CSS, JS <br/>
 - Server: Tomcat <br/>
 - IDE: eclipse, VS Code <br/>
 - Database: MySQL <br/>
 - Library: Spring, Tiles, JSTL, AJAX, jQuery, JDBC, Gson, MyBatis, JSON <br/>
